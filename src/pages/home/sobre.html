Sobre o IF Capivari IF Capivari (Release 0.0.9)

Funcionalidades:

- Disponibilizar artigos e notícias do IFSP.

- Acesso ao Moodle para alunos (notas e disciplinas) e docentes (disciplinas, alunos e notas).

- Mural com login social para procurar/oferecer caronas/moradias.

- Ver posts do IF no Facebook.

 Projeto

Originado como uma proposta de TCC, o projeto visa desenvolver um aplicativo móvel dedicado a solucionar uma demanda do IFSP Campus Capivari. Essa demanda consiste em disponibilizar para a comunidade conteúdos acadêmicos e institucionais em dispositivos móveis. Este projeto vem como uma continuação aproveitando o projeto da API Rest desenvolvida pelo meu amigo Lucas Campaci, a qual alimenta este aplicativo com os dados do câmpus.

Orientado: Douglas Diez Goncalves

 Desenvolvimento

Para desenvolver a proposta, foi escolhida uma abordagem de desenvolvimento mobile híbrido, aonde o código web é compilado em um pacote que roda como uma página web no WebView nativo dos smartphones, e através de plugins e APIs deste pacote, pode acessar o hardware e recursos nativos. Principais tecnologias e ferramentas:

Angular 4 -- Ionic 3

AngularFire + Firebase

Apache Cordova -- NodeJS

Github -- JavaScript

Visual Studio Code
